# Mockmate

Mockmate is an **AI-powered mock interview platform** that helps students and professionals practice interviews, get real-time AI-generated questions, and receive detailed feedback. It’s designed to simulate real-world interview environments and improve preparation.

---

## 🚀 Features

* **AI-Generated Interview Questions** – Get personalized questions based on job role and description.
* **Mock Interview Management** – Save, review, and manage multiple interview sessions.
* **Real-Time Feedback** – Receive instant AI-driven feedback on your responses.
* **User Authentication** – Secure login/signup with Firebase Authentication.
* **Database Storage** – PostgreSQL for structured and scalable data storage.
* **Feedback System** – Users can submit and view feedback for continuous improvement.

---

## 🏗️ Tech Stack

**Frontend**

* React
* Tailwind CSS

**Backend**

* Node.js + Express.js
* PostgreSQL
* Firebase Authentication
